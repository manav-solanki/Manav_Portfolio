Complaint Management System

Manage and track complaints efficiently through a centralized web application.

Project Overview

Complaint Management System is a full-stack web application that enables users to register their complaints while allowing administrators to manage, track, and resolve them efficiently through an intuitive dashboard.

Tech Stack

React.js Node.js Express.js MySQL(Xampp)

Project Highlights

Role: Full Stack Developer
Duration: 3 Months
Project Type: Internship Project
Problem Solved: Streamlined complaint registration, tracking, and management through a centralized platform.
Key Learning: Improved skills in React.js, Node.js, API integration, authentication, and database management.

Key Features

  • Secure User Authentication
  • Complaint Registration and Tracking
  • Admin Dashboard
  • Search and Filter Complaints
  • Complaint Report Generation
  • Dark Mode Support
